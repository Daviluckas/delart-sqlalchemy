# Delart Estética Automotiva

Aplicação web em Flask + SQLAlchemy para cadastramento de agendamentos de serviços
de estética automotiva: **lavagem**, **polimento** e **restauração**.

## Como rodar

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

Acesse: http://127.0.0.1:5000
